# 2. 터미널 사용법

[← 목차로 돌아가기](../../README-KR.md)

---

터미널은 텍스트 명령어로 컴퓨터를 제어하는 프로그램입니다. **이 가이드의 대부분 작업은 터미널에서 진행됩니다.**

> **걱정하지 마세요!** 복잡해 보이지만, 이 가이드에 있는 명령어를 복사해서 붙여넣기만 하면 됩니다.

---

## Windows에서 터미널 열기

### 방법 1: 검색으로 열기 (가장 쉬움) ⭐ 추천

1. 키보드에서 `Windows 키` (⊞) 누르기
2. "PowerShell" 입력
3. **Windows PowerShell** 클릭

```
┌──────────────────────────────────────┐
│  🔍 powershell                       │
│  ─────────────────────────────────── │
│  📘 Windows PowerShell    ← 이것 클릭│
│  📘 Windows PowerShell ISE           │
└──────────────────────────────────────┘
```

<!-- 스크린샷: Windows 검색에서 PowerShell 검색 화면 -->

### 방법 2: 단축키로 열기

1. `Win + R` 키 동시에 누르기 (실행 창이 열림)
2. `powershell` 입력
3. `Enter` 키 누르기

```
┌─ 실행 ─────────────────────────────┐
│  열기(O): powershell               │
│           [확인] [취소] [찾아보기] │
└────────────────────────────────────┘
```

<!-- 스크린샷: 실행 창에 powershell 입력 화면 -->

### 방법 3: 시작 메뉴에서 열기

1. 화면 왼쪽 하단 **시작 버튼** (⊞) **우클릭**
2. **Windows PowerShell** 또는 **Windows 터미널** 선택

---

## Mac에서 터미널 열기

### 방법 1: Spotlight로 열기 (가장 쉬움) ⭐ 추천

1. `Cmd + Space` 키 동시에 누르기 (Spotlight 검색창 열림)
2. "터미널" 또는 "terminal" 입력
3. **터미널** 앱 클릭 (또는 Enter)

```
┌──────────────────────────────────────┐
│  🔍 터미널                           │
│  ─────────────────────────────────── │
│  ⬛ 터미널        응용 프로그램      │ ← 이것 클릭
└──────────────────────────────────────┘
```

<!-- 스크린샷: Spotlight에서 터미널 검색 화면 -->

### 방법 2: Finder에서 열기

1. Dock에서 **Finder** (웃는 얼굴 아이콘) 클릭
2. 상단 메뉴에서 **이동** → **유틸리티** 클릭
3. **터미널** 더블클릭

### 방법 3: Launchpad에서 열기

1. Dock에서 **Launchpad** (로켓 아이콘) 클릭
2. 상단 검색창에 "터미널" 입력
3. **터미널** 앱 클릭
