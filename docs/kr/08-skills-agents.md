[← 목차로 돌아가기](../../README-KR.md)

---

Skills과 Agents는 Claude Code의 기능을 확장하는 도구입니다.

> **이 문서의 목적**: 연구자, HR팀 등 비개발자가 실무에서 활용할 수 있는 기능들을 중심으로 설명합니다.

---

## Skills이란?

Skills은 Claude Code가 **특정 작업을 더 잘 수행**할 수 있도록 도와주는 추가 기능입니다.

```
┌─────────────────────────────────────────────────────────────┐
│  💡 Skills 작동 방식                                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Skills은 Claude Code에 기본 내장되어 있습니다.                 │
│  별도 설치가 필요 없습니다!                                    │
│                                                             │
│  작동 방식:                                                  │
│  • 자동 활성화: 요청 내용에 따라 자동으로 적용됨                  │
│  • 수동 활성화: /명령어로 직접 호출                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 📊 비개발자를 위한 실무 활용 Skills

### Excel/스프레드시트 작업 (xlsx)

연구자와 HR팀이 가장 많이 활용할 수 있는 기능입니다.

| 기능 | 설명 | 예시 요청 |
|------|------|-----------|
| 새 엑셀 파일 생성 | 데이터를 엑셀로 저장 | "이 데이터를 엑셀 파일로 만들어줘" |
| 기존 파일 분석 | 엑셀 데이터 읽고 분석 | "이 엑셀 파일을 분석해줘" |
| 수식 자동 생성 | 복잡한 수식 작성 | "VLOOKUP 수식을 만들어줘" |
| 데이터 정리 | 중복 제거, 포맷 통일 | "중복된 데이터를 정리해줘" |
| 피벗 테이블 | 요약 테이블 생성 | "매출 데이터를 월별로 요약해줘" |
| CSV 변환 | 다양한 형식 변환 | "CSV를 엑셀로 변환해줘" |

**실무 예시 - 연구자:**
```
> survey_results.xlsx 파일을 읽고 응답자 연령대별 평균 점수를 계산해줘
  → xlsx skill이 파일을 분석하고 통계 테이블 생성

> 이 데이터로 상관관계 분석을 하고 결과를 새 시트에 저장해줘
  → 통계 분석 후 결과 저장

> 연구 데이터를 SPSS에서 쓸 수 있는 형식으로 변환해줘
  → CSV 또는 호환 형식으로 변환
```

**실무 예시 - HR팀:**
```
> 직원 명단에서 부서별 인원수를 집계해줘
  → 피벗 테이블 형태로 집계 결과 생성

> 급여 데이터에서 평균, 중간값, 표준편차를 계산해줘
  → 통계 분석 및 결과 정리

> 출퇴근 기록을 월별 근무시간 보고서로 만들어줘
  → 데이터 가공 후 보고서 형식으로 정리

> 두 개의 엑셀 파일을 사번 기준으로 병합해줘
  → VLOOKUP/병합 작업 수행
```

### Word 문서 작업 (docx)

보고서, 제안서, 계약서 등 문서 작업에 활용합니다.

| 기능 | 설명 | 예시 요청 |
|------|------|-----------|
| 새 문서 생성 | Word 문서 작성 | "보고서를 Word로 만들어줘" |
| 기존 문서 편집 | 내용 수정/추가 | "이 문서에 요약을 추가해줘" |
| 서식 적용 | 제목, 표, 목차 등 | "표를 추가하고 서식을 정리해줘" |
| 템플릿 활용 | 양식에 맞게 작성 | "이 양식에 맞춰 작성해줘" |
| 변경 내역 추적 | 수정 사항 기록 | "변경 내역을 추적해서 수정해줘" |

**실무 예시:**
```
> 회의록을 작성해줘. 참석자: 홍길동, 김철수. 안건: 2분기 계획
  → 회의록 템플릿에 맞춰 문서 생성

> 이 연구 결과를 논문 형식의 보고서로 작성해줘
  → 서론, 방법론, 결과, 결론 구조로 작성

> 직원 평가서 양식을 만들어줘
  → 평가 항목이 포함된 Word 양식 생성
```

## 📝 문서 공동 작성 (doc-coauthoring)

체계적인 문서 작성을 도와주는 워크플로우입니다.

**활용 상황:**
- 연구 보고서 작성
- 제안서/기획서 작성
- 정책 문서 작성
- 매뉴얼/가이드 작성

**작동 방식:**
```
1. 문서의 목적과 대상 독자 정의
2. 구조(목차) 설계
3. 섹션별 초안 작성
4. 리뷰 및 수정
5. 최종 검토
```

**예시:**
```
> 신규 직원 온보딩 가이드를 작성해줘
  → doc-coauthoring이 단계별로 문서 작성 안내
```

```
┌─────────────────────────────────────────────────────────────┐
│  💡 Agents 작동 방식                                         │

│                                                             │

│                                                             │
│  Claude Code가 필요에 따라 자동으로 적절한 Agent를               │
```

## 비개발자를 위한 주요 Agents

| Agent | 역할 | 활용 예시 |
|-------|------|----------|
| `librarian` | 문서/연구 자료 분석 | "이 자료들을 분석해줘" |
| `document-writer` | 문서 작성 | "보고서를 작성해줘" |
| `explore` | 빠른 파일 검색 | "관련 파일을 찾아줘" |
| `multimodal-looker` | 이미지/차트 분석 | "이 차트를 분석해줘" |
| `planner` | 작업 계획 수립 | "이 프로젝트 계획을 세워줘" |

**연구자/HR 실무 예시:**
```
> 이 5개의 논문 PDF를 분석해서 핵심 내용을 비교해줘
  → librarian이 문헌 분석 수행

> 설문 결과 차트 이미지를 보고 인사이트를 정리해줘
  → multimodal-looker가 시각 자료 분석

> 채용 프로세스 개선 계획을 세워줘
  → planner가 단계별 계획 수립
```

## Skills 커스텀 폴더 (고급)

나만의 Skills을 추가하고 싶다면 아래 폴더에 파일을 추가하세요:

| OS | 경로 |
|----|------|
| Windows | `C:\Users\사용자명\.claude\skills\` |
| Mac | `~/.claude/skills/` |

**폴더 생성 명령어:**

```powershell
# Windows (PowerShell)
New-Item -ItemType Directory -Force -Path "$env:USERPROFILE\.claude\skills"
```

```bash
# Mac (터미널)
mkdir -p ~/.claude/skills
```

> **참고**: 커스텀 Skills 설정은 고급 사용자를 위한 것입니다. 처음에는 기본 내장 Skills만 사용해도 충분합니다!

## 💡 효과적인 요청 팁

### 좋은 요청의 조건

1. **구체적으로 요청하기**
   ```
   ❌ "데이터 분석해줘"
   ✅ "sales_2024.xlsx의 월별 매출 추이를 분석하고 막대 그래프로 시각화해줘"
   ```

2. **원하는 결과물 명시하기**
   ```
   ❌ "직원 정보 정리해줘"
   ✅ "직원 정보를 부서별, 직급별로 분류해서 피벗 테이블로 만들어줘"
   ```

3. **파일 형식 지정하기**
   ```
   ❌ "보고서 만들어줘"
   ✅ "분석 결과를 Word 문서로 작성하고, 데이터는 별도 Excel 파일로 저장해줘"
   ```

### 복잡한 작업 요청 시

```
> 다음 작업을 순서대로 해줘:
> 1. survey_data.xlsx 파일을 읽어줘
> 2. 연령대별 응답 평균을 계산해줘
> 3. 결과를 막대 그래프로 시각화해줘
> 4. 분석 결과를 보고서로 작성해줘
```
