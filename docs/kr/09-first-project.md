# 9. 첫 프로젝트 시작하기

[← 목차로 돌아가기](../../README-KR.md)

---

이제 모든 설정이 완료되었습니다! 첫 프로젝트를 만들어봅시다.

---

## Step 1: 프로젝트 폴더 만들기

### Windows (PowerShell)

```powershell
# 문서 폴더로 이동
cd $env:USERPROFILE\Documents

# 새 프로젝트 폴더 생성
mkdir my-first-project

# 폴더로 이동
cd my-first-project
```

### Mac (터미널)

```bash
# 문서 폴더로 이동
cd ~/Documents

# 새 프로젝트 폴더 생성
mkdir my-first-project

# 폴더로 이동
cd my-first-project
```

---

## Step 2: Claude Code 실행

```bash
claude
```

---

## Step 3: AI에게 요청하기

```
> 간단한 To-Do 앱을 만들어줘.
> 할 일 추가, 삭제, 완료 체크 기능이 필요해.
```

Claude가 자동으로:
1. 필요한 파일들을 생성하고
2. 코드를 작성하고
3. 실행 방법을 알려줍니다!

---

## Step 4: 결과 확인

생성된 파일을 IDE에서 열어보세요:

**VS Code로 열기:**
```bash
code .
```

또는 VS Code 실행 → **파일** → **폴더 열기** → 프로젝트 폴더 선택

---

## 유용한 프롬프트 예시

### 🌐 웹사이트 생성
```
회사 소개 랜딩 페이지를 만들어줘.
회사명은 "ABC Corp"이고,
서비스 소개, 팀 소개, 연락처 섹션이 필요해.
반응형으로 모바일에서도 잘 보이게 해줘.
```

### 📊 데이터 분석 도구
```
CSV 파일을 업로드하면 데이터를 시각화해주는
간단한 대시보드를 만들어줘.
막대 그래프와 원형 그래프가 있으면 좋겠어.
```

### 🤖 업무 자동화
```
엑셀 파일에서 특정 조건의 데이터만 추출해서
새 파일로 저장하는 파이썬 스크립트를 만들어줘.
조건은 "상태" 열이 "완료"인 행만 추출하는 거야.
```

### 📝 문서 생성
```
이 프로젝트의 README.md 파일을 작성해줘.
프로젝트 설명, 설치 방법, 사용법이 포함되어야 해.
```
