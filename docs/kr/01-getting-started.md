## 용어집

이 가이드에서 사용되는 기본 용어를 설명합니다. **처음 보는 용어가 나오면 이 표를 참고하세요.**

| 용어 | 쉬운 설명 | 비유 |
|------|----------|------|
| **IDE** | 코드를 작성하는 프로그램 | 워드프로세서의 코드 버전 |
| **터미널** | 텍스트 명령어로 컴퓨터를 제어하는 프로그램 | 컴퓨터와 대화하는 채팅창 |
| **CLI** | 텍스트로 프로그램을 실행하는 방식 | 버튼 대신 글자로 명령 |
| **npm** | 프로그램 설치 도구 | 앱스토어 같은 것 |
| **API** | 프로그램 간 통신 규약 | 프로그램끼리 대화하는 언어 |
| **API 키** | AI 서비스 사용을 위한 인증 코드 | 비밀번호 같은 열쇠 |
| **환경 변수** | 컴퓨터 전체에서 사용하는 설정값 | 컴퓨터의 기억 메모 |
| **LiteLLM** | 여러 AI 모델을 통합해주는 서비스 | AI 통합 관제탑 |
| **폴더/디렉토리** | 파일을 담는 공간 | 서류 보관함 |
| **경로(Path)** | 파일이나 폴더의 위치 | 주소 |

## 시스템 요구사항

### Windows
- **운영체제**: Windows 10 (64비트) 이상
- **메모리**: 최소 8GB RAM (16GB 권장)
- **저장공간**: 최소 5GB 여유 공간
- **인터넷**: 안정적인 인터넷 연결 필수

### Mac
- **운영체제**: macOS 10.15 (Catalina) 이상
- **칩**: Intel 또는 Apple Silicon (M1/M2/M3/M4)
- **메모리**: 최소 8GB RAM (16GB 권장)
- **저장공간**: 최소 5GB 여유 공간
- **인터넷**: 안정적인 인터넷 연결 필수
