# 11. FAQ

---

## Q: Do I have to install an IDE?
**A**: It's not required. You can use Claude Code with just the terminal. However, using an IDE:
- Makes it easier to view generated code
- Allows you to see file structure at a glance
- Makes code editing more convenient

---

## Q: Which should I use: VS Code, Cursor, or Antigravity?
**A**:
| IDE | Recommended For |
|-----|-----------------|
| VS Code | Beginners, those who want free options |
| Cursor | Those who use AI coding assistants heavily |
| Antigravity | Those who want autonomous AI agent coding |

---

## Q: What should I do if my API key is exposed?
**A**: Contact the IT team **immediately**!
1. Request to revoke existing key
2. Get new key issued
3. Update environment variables

---

## Q: Which should I use: Claude Code or OpenCode?
**A**:
| Tool | Features | Recommended |
|------|----------|-------------|
| Claude Code | Official Anthropic, stable, feature-rich, supports Windows/Mac | General use |
| OpenCode | Open source, supports various AI models, **Mac only** | Testing multiple models on Mac |

**For beginners, we recommend Claude Code!**
**Windows users can only use Claude Code.**

---

## Q: What should I do when an error occurs?
**A**:
1. **Copy** the error message
2. Ask Claude Code directly:
   ```
   > This error occurred: [paste error message]
   ```
3. If not resolved → Contact Jason

---

## Q: Can I use it without internet?
**A**: No, **internet is required** to connect to AI services.

---

## Q: Can I use it outside the office?
**A**: It depends on the LiteLLM server access policy.
- VPN requirement
- External access permission
→ Check with IT team.

---

## Q: How do I run the generated code?
**A**: It depends on the code type:
- **HTML files**: Open in browser
- **Python files**: `python filename.py`
- **JavaScript files**: `node filename.js`
- **React apps**: Claude Code will tell you how to run it

---

## Q: Do I need to install Skills or Agents separately?
**A**: No! They are all **built into Claude Code by default**. You can use them right away without separate installation.
