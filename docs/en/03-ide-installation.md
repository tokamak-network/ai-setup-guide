## VS Code Installation (Free) ‚≠ê Recommended

### Windows

1. Go to **https://code.visualstudio.com** in your web browser
2. Click the blue **Download for Windows** button
3. Double-click the `VSCodeUserSetup-x64-*.exe` file in your Downloads folder
4. Select "I agree" ‚Üí Click **Next**
5. Confirm installation path ‚Üí Click **Next**
6. **‚ö†Ô∏è Important**: **Check all** the options below:

```
‚òëÔ∏è "Add to PATH" (Must check!)
‚òëÔ∏è Add "Open with Code" action to file context menu
‚òëÔ∏è Add "Open with Code" action to directory context menu
```

7. Click **Install** ‚Üí Wait for completion
8. Click **Finish**

### Mac

1. Go to **https://code.visualstudio.com** in your web browser
2. Click the **Download for Mac** button

**How to check your Mac chip:**
```
1. Click the  icon at the top left of the screen
2. Click "About This Mac"
3. Check "Chip" or "Processor":
   - "Apple M1/M2/M3/M4" ‚Üí Choose Apple Silicon
   - "Intel" ‚Üí Choose Intel
```

3. Double-click the downloaded `.zip` file (auto-extracts)
4. Drag `Visual Studio Code` app to the **Applications** folder
5. Double-click **Visual Studio Code** in the **Applications** folder
6. "App downloaded from the internet" warning ‚Üí Click **Open**

**Setting up `code` command in Terminal (Optional):**
1. Run VS Code
2. Press `Cmd + Shift + P` simultaneously
3. Type "shell command"
4. Select **Shell Command: Install 'code' command in PATH**

## Cursor Installation (Built-in AI Features)

### Windows

1. Go to **https://cursor.com** in your web browser
2. Click **Download for Windows**
3. Double-click the downloaded `Cursor Setup *.exe` file
4. Follow the installation wizard
5. Run after completion

### Mac

1. Go to **https://cursor.com** in your web browser
2. Click **Download for Mac**
3. Double-click the downloaded `.dmg` file
4. Drag the Cursor icon to the **Applications** folder
5. Run from Applications

---

## Antigravity Installation (AI Agent-based)

> **Note**: Antigravity is an IDE where AI agents autonomously assist with coding.

### Windows

1. Go to **https://antigravity.dev** in your web browser
2. Click the **Download** button
3. Download the Windows installation file
4. Double-click the downloaded `.exe` file
5. Follow the installation wizard
6. Run after completion

### Mac

1. Go to **https://antigravity.dev** in your web browser
2. Click the **Download** button
3. Select and download for Mac (Intel or Apple Silicon)
4. Double-click the downloaded `.dmg` file
5. Drag Antigravity to the **Applications** folder
6. Run from Applications
