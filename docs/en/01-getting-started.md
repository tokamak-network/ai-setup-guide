## Glossary

Here are the basic terms used in this guide. **Refer to this table when you encounter unfamiliar terms.**

| Term | Simple Explanation | Analogy |
|------|-------------------|---------|
| **IDE** | A program for writing code | Code version of a word processor |
| **Terminal** | A program that controls the computer via text commands | A chat window for talking to your computer |
| **CLI** | Running programs via text commands | Commands with letters instead of buttons |
| **npm** | A program installation tool | Like an app store |
| **API** | Communication protocol between programs | A language for programs to talk to each other |
| **API Key** | Authentication code for using AI services | A key like a password |
| **Environment Variable** | Settings used across the entire computer | Computer's memory notes |
| **LiteLLM** | A service that integrates multiple AI models | AI integration control tower |
| **Folder/Directory** | A space to store files | A filing cabinet |
| **Path** | The location of a file or folder | An address |

## System Requirements

### Windows
- **Operating System**: Windows 10 (64-bit) or later
- **Memory**: Minimum 8GB RAM (16GB recommended)
- **Storage**: Minimum 5GB free space
- **Internet**: Stable internet connection required

### Mac
- **Operating System**: macOS 10.15 (Catalina) or later
- **Chip**: Intel or Apple Silicon (M1/M2/M3/M4)
- **Memory**: Minimum 8GB RAM (16GB recommended)
- **Storage**: Minimum 5GB free space
- **Internet**: Stable internet connection required
